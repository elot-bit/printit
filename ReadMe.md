# Projet Site Web Print-It 

## Description du projet (carrousel)

- Création d'un carrousel de photos et de textes en bannière d'un site Web
- Modification des codes HTML, CSS et JavaScript d'un site Web
 
## Objectif du carrousel

- Les images se succèdent et tournent à l'infinie dans les deux sens
- Les flèches font tourner les images dans des ordres opposés
- Les dots sont surlignés en fonction de l'image exposée
- Les textes changent en même temps que les images -

## Fonctionnalités

- Flèches
- Points (dots)
- Boucle infinie
- Texte dynamique

## Technologies utilisées
- HTML
- CSS
- JavaScript
